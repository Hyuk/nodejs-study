# Importing npm module

* 노드 버전 확인하기
```bash
$ node -v
```

* npm 버전 확인하기
```bash
$ npm -v
```

* npm 초기화
```bash
$ npm init
```

* npm 초기화 with Default value
```bash
$ npm init -y
```

* package.json file 이 자동 생성된다.
```json
{
  "name": "04-importing-npm-module",
  "version": "1.0.0",
  "description": "",
  "main": "app.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}

```
